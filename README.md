# Snow_White_Week5
המשחק "שלגייה" הוא משחק טקסט אינטראקטיבי שנבנה באמצעות Twine, ומתאר את מסעה של שלגייה מרגע בריחתה מהמלכה הרעה ועד להחלטות גורליות שיקבעו את עתידה. העלילה מתפצלת במספר רב של נקודות בחירה, שבהן השחקן בוחר כיצד שלגייה תפעל להסתתר, לברוח, לתת אמון בדמויות, או לחקור את העולם שסביבה. לאורך הדרך ניתן להיפגש עם דמויות מוכרות כמו הצייד, הגמדים והזקנה המסתורית. חלק מהבחירות מובילות לנתיבים שונים אך מתאחדות מחדש לנקודות מרכזיות, כדי לשמור על משחק זורם. המטרה של שלגיה זה להגיע לנסיך ולבחור בחירות אמיצות.
